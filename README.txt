# India CPI Inflation Analysis using Excel

## Objective
The objective of this project is to analyze **Consumer Price Index (CPI) inflation trends in India**
and derive meaningful economic insights using Microsoft Excel through structured data analysis
and dashboard-driven storytelling.

## Dataset
The dataset contains India’s CPI inflation data across multiple years, categories, and sectors
(Rural, Urban, and Combined).  
The raw data was provided in **Excel format** and cleaned, transformed, and analyzed for insights.

## Tools Used
- Microsoft Excel

## Key Excel Techniques Applied
- Data cleaning and preprocessing
- Pivot Tables and Pivot Charts
- VLOOKUP / XLOOKUP
- Conditional Formatting
- Month-on-Month (MoM) and Year-on-Year (YoY) analysis
- Correlation analysis
- Interactive dashboard design

## Analysis Performed
- CPI inflation trend analysis across years
- Category-wise inflation comparison
- Rural vs Urban inflation behavior
- Month-on-Month and Year-on-Year inflation changes
- Impact analysis of the COVID-19 pandemic on inflation
- Contribution of different categories to overall CPI
- Relationship between CPI categories and crude oil prices

## Key Insights
- CPI inflation shows a **long-term upward trend**, with noticeable fluctuations during crisis periods
- Food & Beverages and Transport categories contribute significantly to inflation volatility
- COVID-19 had a visible impact on inflation behavior across multiple categories
- Rural and Urban inflation patterns differ during peak inflation periods
- Certain CPI categories show moderate correlation with crude oil price movements

## Conclusion
This project demonstrates the use of Excel as a powerful analytical tool for
economic and inflation analysis, highlighting skills in data preparation,
trend analysis, and dashboard-based insight communication.
